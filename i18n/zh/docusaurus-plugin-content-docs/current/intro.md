# 欢迎使用 Wiki Truth 文档

![Wiki Truth Banner](/docs/12-md.jpg)

**Wiki Truth** 是全球首个**去中心化吹哨人激励和犯罪证据市场 (Decentralized Whistleblower Incentive and Crime Evidence Marketplace)**，基于 SEC 举报人计划 (SEC Whistleblower Program) 的 Web3 数字化升级。

通过Wiki Truth，吹哨人可以将**有风险真相 (Risk Truth)** 转化为链上资产（Truth Box），通过交易市场变现，获得经济回报，从而激发更多人参与正义行动的积极性。
